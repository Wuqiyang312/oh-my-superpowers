# 分类标签

Skills 使用五种标准的分类角色。本文档将这些角色映射到本仓库问题追踪器中实际使用的标签字符串。

| oh-my-superpowers 中的标签 | 本追踪器中的标签 | 含义 |
| `needs-triage`             | `needs-triage`   | 维护者需要评估此问题 |
| `needs-info`               | `needs-info`     | 等待报告者提供更多信息 |
| `ready-for-agent`          | `ready-for-agent`| 已完全明确，可供 AFK agent 处理 |
| `ready-for-human`          | `ready-for-human`| 需要人工实现 |
| `wontfix`                  | `wontfix`        | 不处理 |

当某个 Skill 提及一个角色（例如"应用 AFK-ready 分类标签"）时，请使用本表格中对应的标签字符串。

编辑右侧列以匹配你实际使用的术语。
