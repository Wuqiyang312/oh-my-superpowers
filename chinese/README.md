# Chinese — 中文特定技能

面向中国开发者的专属规范和工作流技能。

| 技能 | 描述 |
|------|------|
| [chinese-code-review](./chinese-code-review/SKILL.md) | 中文代码审查规范 |
| [chinese-commit-conventions](./chinese-commit-conventions/SKILL.md) | 中文提交规范 |
| [chinese-documentation](./chinese-documentation/SKILL.md) | 中文技术文档编写规范 |
| [chinese-git-workflow](./chinese-git-workflow/SKILL.md) | 中文 git 工作流（Gitee/Coding/极狐 GitLab 适配） |

## 使用场景

- 项目中有中文注释、中文 README → 启用中文系列技能
- commit 历史中有中文 → 使用中文提交规范
- 用户用中文交流 → 优先考虑中国特色技能

中国特色技能与通用技能**叠加使用**，不互斥。
